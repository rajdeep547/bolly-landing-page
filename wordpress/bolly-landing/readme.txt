Recreates the Bolly shampoo landing page with an interactive 3D product bottle.

== Description ==

This plugin provides:

* Full landing page hero matching the Bolly reference design
* Interactive 3D shampoo bottle (drag on desktop, swipe on mobile)
* Elementor widgets: Bolly Landing Hero and Bolly 3D Bottle
* Shortcodes: `[bolly_landing]` and `[bolly_3d_bottle]`
* Automatic landing page creation on activation

== Installation ==

1. Upload the `bolly-landing` folder to `/wp-content/plugins/`
2. Activate the plugin
3. Install and activate Elementor
4. Open the auto-created "Bolly Landing" page or add the Bolly Landing Hero widget in Elementor

== Frequently Asked Questions ==

= Does this require Elementor? =

Elementor is required for the widget-based workflow. Shortcodes work without Elementor.

= How do users rotate the bottle? =

Drag with the mouse on desktop or swipe on touch devices.
